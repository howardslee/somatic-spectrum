# somatic-spectrum
GenomicSEM analysis of shared genetic architecture across fibromyalgia, chronic pain, migraine, ME/CFS, and IBS. Single latent factor maps to prenatal neurodevelopmental genes, cerebellar enrichment, and first-trimester neuroblast cell types.

See [GenomicSEM/README.md](GenomicSEM/README.md) for the reproducible analysis command sequence.
